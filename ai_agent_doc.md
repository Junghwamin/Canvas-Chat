# AI 에이전트 기술 문서

## 1. 에이전트란 무엇인가?

에이전트(Agent)는 자율적으로 환경을 인식하고, 의사결정을 내리며, 목표를 달성하기 위해 행동하는 시스템입니다.

## 2. Level 3 자율성

Level 3 자율성은 에이전트가 다음과 같은 능력을 갖추는 것을 의미합니다:
- **자율적 계획 수립**: 목표를 분석하고 단계별 계획을 세움
- **도구 선택**: 상황에 맞는 도구를 자동으로 선택
- **오류 복구**: 실패 시 대안을 찾아 재시도

## 3. Chain of Thought (사고의 연쇄)

Chain of Thought는 복잡한 문제를 단계별로 분해하여 해결하는 기법입니다.
예시: "수학 문제 풀기" → "1. 문제 이해 → 2. 공식 선택 → 3. 계산 → 4. 검증"

## 4. RAG의 장점

RAG(Retrieval-Augmented Generation)의 주요 장점:
1. **정확한 정보 제공**: 할루시네이션 감소
2. **최신 정보 활용**: 학습 데이터 이후 정보도 활용 가능
3. **출처 추적 가능**: 어떤 문서에서 정보를 가져왔는지 확인 가능

## 5. Few Shot Learning

Few Shot Learning은 몇 가지 예시만으로 모델의 행동을 유도하는 기법입니다.
적은 예시로도 원하는 출력 형식을 학습할 수 있어 프롬프트 엔지니어링에 효과적입니다.
